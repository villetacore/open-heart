## [Unreleased]

### Added
- **ИИ-генерация текстур** — первый срез направления «нейросеть → ассеты»:
  - `tools/aigen.py` — клиент к серверу генерации (AUTOMATIC1111 `/sdapi/v1/txt2img`
    или ComfyUI `/prompt`), конфиг `tools/aigen.json` (url/бэкенд/модель/шаги);
  - шаблоны промптов по типам ассетов `tools/aigen_templates.json` (из ASSET_GUIDE);
  - автоматическая постобработка через `tools/process_sprites.py`: удаление фона,
    ресайз, сборка кадров, раскладка в `godot/assets/*` (включая маршрутизацию
    `dtile_*`/`sky_*` по подпапкам) — игра подхватывает PNG при следующем запуске;
  - окно **«🎨 ИИ-текстуры»** в редакторе OpenHeart: тип ассета + id + описание →
    кнопка «Сгенерировать», превью результата, генерация в фоне (редактор не виснет).
- **Пауза (Esc)**: меню паузы в Explore-режиме (продолжить / выйти в главное меню);
  бой останавливается в ЛЮБОМ меню — в инвентаре, перках и диалогах враги
  заморожены, снаряды замирают и урон по игроку не проходит (раньше враги били
  игрока, пока открыт инвентарь).
- `cargo test`: тесты контента — все JSON каждого пресета парсятся и проверяются
  на ссылочную целостность (квест → NPC/враг/предмет, спавны карт → enemies/items);
  встроенные фолбэк-копии тоже проверяются.

### Changed
- **DLL перенесена внутрь проекта**: `OpenHeart.gdextension` теперь указывает на
  `res://bin/openheart.dll` (раньше — `res://../rust/target/...`, вне проекта, из-за
  чего экспорт игры был невозможен). `build.bat`/`run.ps1`/`watch.ps1` копируют DLL
  в `godot/bin/` после сборки; папка в `.gitignore`.
- Битые `enemies.json`/`items.json`/`npcs.json`/`quests.json`/`level.json` больше не
  превращаются молча в пустые списки: предупреждение в лог + фолбэк на встроенную
  копию core (как у weapons/classes/perks). Неизвестный id в спавнах тоже логируется.
- Нечитаемый сейв больше не выглядит как «сейва нет»: файл откладывается в
  `user://save.corrupt.json` с предупреждением; версия сейва проверяется при чтении.
- `Esc` больше не «отпускает мышь» — вместо этого открывает меню паузы
  (в инвентаре/перках `Esc` закрывает экран).

### Планируется
- Статусы урона (горение/кровотечение/стан), крит-статы, классовые активки (`F`).
- Звук и музыка (M7), рывок/слайд (M5), 3D-гизмо карт в редакторе.
- Разбиение `game.rs` на подсистемы, workspace `oh_core`/`oh_game`, тесты.
- Полный план: [docs/DESIGN_PLAN.md](docs/DESIGN_PLAN.md).

## [0.4.0] — 2026-07-04

Крупный релиз: пресеты, редактор игры, карты из JSON, вертикальные данжи.

### Added
- **Система пресетов** `godot/presets/<id>/` — каждый пресет = отдельная игра
  (свои оружие/классы/перки/враги/предметы/NPC/квесты/карты + `preset.json`).
  Переключение в главном меню; сейв помнит свой пресет. В комплекте: `core`
  («Неоновое Сердце») и `arena` («Кровавая арена»).
- **Редактор игры** — плагин `godot/addons/oh_editor`, main-screen вкладка «OpenHeart»:
  формы по схемам для всех типов контента, CRUD, сохранение пресета, создание
  пресета-копии, «Замок ядра» (read-only на фундаментальные файлы).
- **Карты из JSON** (`maps/*.json`, `rust/src/map.rs`): фигуры box/ramp/stairs/cylinder,
  здания с неон-вывесками, пропсы, свет, светящиеся неон-каналы, спавны, врата данжа,
  окружение на карту (небо/туман/амбиент). Новый многоярусный хаб; карта арены.
- **Данжи 2.0**: комнаты на разных высотах (0/0.8/1.6), переменные потолки, ступени
  на границах уровней, платформы; **миникарта данжа** (floor_map → текстура + точка игрока).
- **NPC и квесты data-driven** (`npcs.json`, `quests.json`): гиверы с автогенерируемыми
  диалогами (взять/прогресс/сдать) и цепочками; трекинг kill/collect/clear_dungeon.
- Контент-пак: **14 типов врагов** (варианты по спрайту/масштабу с тематическими
  резистами), 18 предметов, 18 перков, 8 синергий, 12 NPC, 7 квестов.
- Постобработка (виньетка + хроматическая аберрация + сканлайны), glow/bloom,
  ACES-тонмаппинг, HUD-иконки HP/боезапаса.
- Поля `sprite`/`scale` у врагов; `Effect::Xp` в диалогах.

### Changed
- Рендерер → **Forward Mobile (Vulkan)**; анизотропная фильтрация текстур,
  аттенюация света.
- Данные переехали из `godot/data/` в `godot/presets/core/`; загрузчики контента
  стали перезагружаемыми (смена пресета без перезапуска процесса).
- Спавны мира берутся из карты пресета (`level.json` остаётся фолбэком).

### Fixed
- Пустой `npcs.json` больше не подтягивает legacy-NPC (различаются «файла нет» и
  «пустой список»).
- Спавны арены отодвинуты от точки появления игрока (мгновенная смерть на старте).
- Имя локации в HUD берётся из карты, а не из захардкоженной строки.

## [0.3.0] — 2026-07-04

### Added
- **Дерево перков** (3 ветки, ранги, требования) и **синергии** (комбо-эффекты);
  очки за уровни; экран прокачки (`P`); персист в сейве.
- **Типы урона** (физический/огонь/энергия/пустота) и **резисты врагов** — выбор
  оружия обретает тактический смысл.
- Data-driven оружие и классы (`weapons.json`, `classes.json`) с фолбэком на
  встроенные копии при битом JSON.
- Смерть → возврат в хаб с потерей 25 % золота (вместо стирания сейва).

### Changed
- UI масштабируется под любое окно (`canvas_items`-стретч, база 1920×1080) —
  починены клики меню на не-1920 экранах.

## [0.2.0] — 2026-07-03

### Added
- Переход к 3D от первого лица: открытый мир (хаб + пустоши), процедурный данж
  с боссом и порталами, 3 класса × 3 спека, 8 стволов с FP-спрайтами,
  hitscan/мили/снаряды, XP/уровни, HUD, сейвы v2.
- Пайплайн нарезки ассетов из ChatGPT-атласов (`tools/slice_*.py`):
  оружие, пикапы, эффекты, UI, небо, 77 тайлов данжа, 35 пропсов.
