# OpenHeart

MVP визуальной новеллы / top-down RPG на **Redot** (форк Godot 4.x).
**Почти вся логика — на Rust** через [`gdext` / godot-rust](https://github.com/godot-rust/gdext) (GDExtension).
GDScript не используется: сцена содержит один узел с нативным Rust-классом `Game`, который строит весь мир, UI, NPC и системы.

## Что уже работает (MVP)

- 🚶 **Ходьба** top-down (WASD / стрелки), камера следует за игроком.
- 🗣️ **Диалоги** — подходишь к NPC, жмёшь `E`, листаешь реплики.
- 👤 **Система персонажа** — имя, уровень, опыт (с авто-левел-апом), HP, атака, золото.
- 🎒 **Инвентарь** — предметы со стопками (клавиша `I`).
- 📜 **Квесты** — журнал с активными/выполненными (клавиша `J`).
- 🗺️ **Локации** — Деревня и Лес, индикатор текущей локации в HUD.
- 🧩 **NPC с ролями** — Старейшина (квестодатель), Торговка (выдаёт зелья), Стражник (флавор).
- 🌿 **Предмет в мире** — целебный корень в лесу; подбирается при подходе и сдаётся в квесте.

### Управление

| Клавиша            | Действие                         |
|--------------------|----------------------------------|
| `W A S D` / стрелки | движение                        |
| `E` / `Space` / `Enter` | взаимодействие / далее в диалоге |
| `I`                | инвентарь                        |
| `J` / `Q`          | журнал квестов                   |

### Готовый игровой цикл квеста

1. Подойди к **Старейшине** (деревня, слева) → получишь квест «Целебный корень».
2. Иди вправо в **Лес** → подбери зелёный корень.
3. Вернись к Старейшине → сдай квест: **+80 опыта, +25 золота** (и, скорее всего, новый уровень).

## Структура

```
OpenHeart/
├─ rust/                  # вся игровая логика (Rust, cdylib)
│  ├─ Cargo.toml
│  └─ src/
│     ├─ lib.rs           # точка входа GDExtension
│     ├─ game.rs          # корневой узел-«мозг»: мир, UI, диалоги, циклы
│     ├─ player.rs        # игрок (CharacterBody2D, движение, отрисовка)
│     ├─ npc.rs           # NPC (имя, роль, цвет)
│     ├─ character.rs     # статы/уровни/опыт
│     ├─ item.rs          # инвентарь и предметы
│     ├─ quest.rs         # квесты и журнал
│     └─ dialogue.rs      # структуры реплик
└─ godot/                 # проект Redot/Godot
   ├─ project.godot
   ├─ OpenHeart.gdextension  # связывает движок с rust/target/.../openheart.dll
   ├─ main.tscn          # одна нода типа Game
   └─ icon.svg
```

> Графика — программные примитивы (цветные прямоугольники), ассеты не нужны.
> Это сделано намеренно для MVP: вся отрисовка тоже в Rust (`draw()`).

## Сборка и запуск

### 1. Собрать Rust-библиотеку

```bash
cd rust
cargo build            # debug → rust/target/debug/openheart.dll
# для релиза: cargo build --release
```

`gdext` компилируется со встроенным API Godot 4.x — **установленный движок для сборки не нужен**.

### 2. Открыть проект в Redot

1. Скачай Redot Engine 4.3+ (https://redotengine.org) — он совместим с GDExtension Godot 4.3.
   (Подойдёт и обычный Godot 4.3+, файл `.gdextension` тот же.)
2. Открой папку `godot/` как проект (выбери `project.godot`).
3. Движок подхватит `OpenHeart.gdextension` и загрузит `openheart.dll`.
4. Нажми **F5** (Run).

> Если меняешь Rust-код: пересобери `cargo build` и перезапусти проект.
> `reloadable = true` в `.gdextension` позволяет горячую перезагрузку при возврате фокуса в редактор.
