# OpenHeart

**2.5D Action-RPG** в духе *DOOM / Forgive Me Father*: полноценный 3D-мир от первого лица,
вся графика — спрайты-биллборды и пиксельные текстуры. Движок — **Redot** (форк Godot 4.x),
**вся логика на Rust** через [`gdext` / godot-rust](https://github.com/godot-rust/gdext) (GDExtension).
GDScript не используется.

## Что работает

- 🌆 **Открытый мир** — площадь-хаб с неоном, NPC и торговцами + пустоши с руинами, врагами и лутом.
- 🕳️ **Процедурные данжи** — сами данжи генерируются заново при каждом входе (комнаты, коридоры,
  темы оформления, спавны, босс, тайники). Карта мира статичная — процедурны только данжи.
- ⚔️ **3 класса × 3 специализации**:
  - **БЕРСЕРК** (мили): пила + энергоклинок — *Кровожнец* (вампиризм), *Таран* (+HP), *Вихрь* (скорость).
  - **ШТУРМОВИК** (клоус-рэнж): пистолет + дробовик — *Экзекутор* (+урон), *Бастион* (+HP), *Дуэлянт* (-КД).
  - **ОПЕРАТОР** (мид-рэнж): автомат + плазма — *Снайпер* (+урон, гвоздемёт), *Подрывник* (ракетница), *Техник* (+боезапас).
- 🔫 **8 стволов** с FP-спрайтами и анимацией выстрела (нарезаны из атласов): энергоклинок, цепная пила,
  пистолет, дробовик, автомат, гвоздемёт, плазмаган, ракетница. Hitscan, мили-удары и летящие снаряды
  (ракета со сплэшем и самоуроном).
- 📈 **Прокачка** — XP за убийства, уровни (+HP, +урон), «Сердца жизни» (+15 макс. HP навсегда).
- 👹 **6 типов врагов** с AI (патруль → погоня → атака), скейлом по глубине данжа; снайпер стреляет
  только при прямой видимости. Босс-«страж» на дне каждого данжа открывает портал глубже.
- 🗣️ **Диалоги, квесты, инвентарь, сохранения** — как и раньше, но теперь с классом и арсеналом (save v2).
- 💥 Эффекты: кровь, взрывы, трейсеры, телепорты, лечение — спрайтами из атласа.

### Управление

| Клавиша | Действие |
|---|---|
| `WASD` | движение |
| `Shift` | спринт |
| `Space` | прыжок |
| `ЛКМ` | огонь (автоматика — удержание) |
| `1–8` / колесо мыши | смена оружия |
| `E` | взаимодействие / диалог / порталы |
| `Q` | быстрое лечение (аптечка из инвентаря) |
| `I` | инвентарь |
| `1–4` | выбор реплики в диалоге |

### Игровой цикл

1. Новая игра → **выбор класса и спеки** (клавиши 1-3).
2. Хаб: NPC, квесты, лут вокруг площади. На севере — **врата данжа** (арка с порталом).
3. Данж: зачищай комнаты, собирай патроны и оружие из тайников, найди **стража** —
   после его смерти открывается портал глубже. Выход — портал в стартовой комнате.
4. Каждый вход в данж — **новая генерация**; глубина растёт, враги толще, лут жирнее.

## Структура

```
OpenHeart/
├─ rust/                  # вся игровая логика (Rust, cdylib)
│  └─ src/
│     ├─ lib.rs           # точка входа GDExtension
│     ├─ game.rs          # оркестратор: режимы, боёвка, HUD, порталы, выбор класса
│     ├─ world.rs         # открытый мир: хаб, пустоши, врата данжа
│     ├─ dungeon.rs       # процедурный генератор данжей (seeded)
│     ├─ classes.rs       # 3 класса × 3 спека, лоадаут, кривая опыта
│     ├─ weapon.rs        # арсенал: 8 стволов, боеприпасы, FP-спрайты
│     ├─ player.rs        # FPS-контроллер (спринт, статы от класса)
│     ├─ enemy.rs         # AI врагов, LOS, скейл по глубине
│     ├─ gfx.rs           # утилиты: боксы, биллборды, свет, RNG, кэш текстур
│     ├─ game_state.rs    # состояние: XP/уровни, флаги, квесты, инвентарь
│     ├─ save.rs          # save v2 (класс, арсенал, сердца, сид данжа)
│     └─ ...              # диалоги, квесты, меню, локаль
├─ godot/                 # проект Redot/Godot
│  ├─ project.godot       # + инпуты: weapon_1..8, sprint, use_med, колесо
│  ├─ data/               # enemies.json (с XP), items.json, level.json (мир)
│  └─ assets/
│     ├─ sprites/weapons_fp/   # 8 стрипов оружия от первого лица (8 кадров)
│     ├─ sprites/pickups/      # патроны ×4, сердце, душа, свиток, граната
│     ├─ sprites/projectiles/  # ракета
│     ├─ sprites/props/        # 35 пропсов: неон-вывески, мебель, улица, ванная
│     ├─ sprites/characters/   # NPC ×8 и враги ×6 (спрайтшиты 512×256)
│     ├─ effects/              # взрыв, кровь, трейсер, телепорт... (8 шт)
│     ├─ ui/                   # иконки HUD (8 шт)
│     └─ textures/             # стены/полы + dungeon/ (77 тайлов) + sky/ (4 неба)
└─ tools/                 # пайплайн нарезки атласов
   ├─ slice_atlases.py    # основная нарезка (оружие, пикапы, эффекты, тайлы)
   ├─ slice_fix.py        # правочный проход (подписи, слипшиеся ячейки)
   ├─ slice_props.py      # пропсы по измеренным боксам
   └─ preview/            # контрольные листы для визуальной проверки
```

## Сборка и запуск

```bash
cd rust
cargo build            # → rust/target/debug/openheart.dll
```

Открой `godot/project.godot` в **Redot 4.3+** (или Godot 4.3+) и жми **F5**.
Либо одной командой: `./run.ps1` (сам соберёт Rust и найдёт движок).

> Меняешь Rust-код → `cargo build` → перезапусти сцену.
> Нарезать ассеты заново: `python tools/slice_atlases.py && python tools/slice_fix.py && python tools/slice_props.py`.
